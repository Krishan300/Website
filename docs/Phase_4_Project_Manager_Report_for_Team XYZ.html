Explain how you are using memcachier to cache documents.

Alex uses the Google drive retrieved ID as the key and caches the contents of the file he is sending to the Google drive. Alex is also using memcachier to replace the functionality of the hashTable from phase 3. Alex created an object that gets connections to the memcachier. The object is used whenever a connection is needed to the memcachier. The object takes a username, password, and server and uses those to create a connection object that connects to the server. The object can be used to put things into the cache and retrive data from the cache. The object is used to abstract the process of connecting to both the file cache and the secretkey cache.
